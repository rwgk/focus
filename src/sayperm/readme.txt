------------
Compilation
------------
For the compilation of 'sayperm' the sginfo package (R.W. Kunstleve) must be linked
into the program. Either the c-code or a library must be available. The makefile
must be changed in order to do a platform dependent configuration.
---------------
Run the program
---------------
The list below shows the names of input files. Example files can be found in the 
./example directory
